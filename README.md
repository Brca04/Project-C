# Project-C
This repository contains the project work from the course [Project C](https://www.fer.unizg.hr/en/course/proc) at FER.
## About
Promoting cooperation between students in a team environment while creating practical solutions to specific problem. Students work in teams of 3 or more students, under the guidance of the lecturer who specifies the project topic. The project requires the students to study the necessary literature, analyse similar problems and solutions, identify project requirements, define technical objectives, plan and manage time, consider alternative solutions, make decisions, implement the selected solution, write technical documentation and present the project results.
## Learning Outcomes
- define project goals
- construct project team
- develop project task
- demonstrate results of project task
- analyze finished project task
- identify problems during development of the project task
